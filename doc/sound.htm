Sound

PyTraffic uses GStreamer (via PyGObject) for sound effects and music playback. On modern Linux systems GStreamer automatically selects the appropriate audio backend — typically PipeWire or PulseAudio — so no manual configuration is needed.

Troubleshooting

If you get no sound, check the following:

  • Make sure your system audio daemon is running. On most distributions this is PipeWire (with the pipewire-pulse compatibility layer) or PulseAudio. You can verify with pactl info.
  • Check that PyTraffic is not muted in your desktop volume mixer (e.g. pavucontrol, pw-top, or the system tray volume applet).
  • Ensure the required GStreamer plugins are installed. On Fedora: dnf install gstreamer1-plugins-base gstreamer1-plugins-good; on Debian/Ubuntu: apt install gstreamer1.0-plugins-good.
  • For additional format support (mp3, flac, etc.) install the extra plugin packages: gstreamer1-plugins-ugly / gstreamer1.0-plugins-ugly.
